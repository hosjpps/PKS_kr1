# ПКС — Контрольная работа №1

## Часть 1: Управление библиотекой книг (WPF + EF Core)

Приложение для управления небольшой библиотекой на **.NET 8**, **WPF** и **Entity Framework Core** (SQLite).

**Функционал:**
- CRUD для книг, авторов и жанров
- Поиск книг по названию
- Фильтрация по автору и жанру
- Отображение количества экземпляров в наличии

**Запуск:**
```bash
cd LibraryManagement
dotnet run
```
